1、MkDocs

1、启动内建服务器：

$ cd myproject
$ mkdocs serve
$ mkdocs serve --dev-addr=127.0.0.1:13809  //  http://localhost:13809 
$ mkdocs serve --dev-addr=127.0.0.1:13808  //  http://localhost:13808

http://localhost:13809

http://localhost:13808

2、站点生成：

$ mkdocs build

安装wine

./configure --enable-win64
